About Quantum Proxy

Quantum Proxy is a lightweight TCP proxy designed to secure long-term proxy connections using Post-Quantum Cryptography (PQC) and hybrid X.509 certificates.

It enables secure mTLS communication through OpenSSL + oqs-provider, supporting both traditional and PQC algorithms via hybrid negotiation.
